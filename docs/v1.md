type RegisterRequest struct {

    Language   *string `json:"language" binding:"omitempty,oneof=zh en"`
}

    OldPassword string `json:"old_password" binding:"required"`

### Service `internal/service/auth_service.go`:

  - bcrypt 哈希密码 (cost=12)
  - 创建用户
  - 生成 access_token + refresh_token

  - 查找用户
  - 验证密码
  - 生成 token 对

  - 解析 refresh_token
  - 验证未过期
  - 生成新 token 对

- UpdateProfile(userId uuid.UUID, req dto.UpdateProfileRequest) (*dto.UserInfo, error)

- PUT  /api/v1/auth/profile    (需认证)

每个 handler 需要：参数校验、调用 service、统一响应格式、Swagger 注解。
将 refresh_token 存入 Redis（key = "refresh:{userId}", TTL = 7天），刷新时验证 Redis 中是否存在。
